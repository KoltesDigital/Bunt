BUNT
====

Submission for the Experimental Gameplay Project, January 2012
"Five buttons"

Usage
-----

Play the game with the keys 1 - 5, on top of your keyboard.
You can also play with keys F1 - F5 or a Xbox controller.
Quit with escape.
